# VPC

## SAA 예상 문제
VPC CIDR이 10.0.0.0/16 이라면 몇 개의 /24 subnet을 만들 수 있는가?

### VPC 란?
나만 사용하는 aws 내부 네트워크
즉 큰 아파트 단지 느낌

### CIDR 10.0.0.0/16 이란?
- ip 주소(10.0.1.15): 32비트 숫자
- /16 : 앞의 16비트는 네트워크 주소. 고정된 주소임.
        -> 16비트인 10.0 까지는 고정이고 뒤는 마음대로 사용가능
```bash
            10.0.1.15/16 -> CIDR
                |        
                ↓
10       0        1        15
00001010.00000000.00000001.00001111
 (8비트)   (8비트)   (8비트)   (8비트)  -> 총 32비트
                 | 
네트워크 부분(16비트) | 호스트 부분 (16비트)                    
```

### /24 subnet 이란?
vpc 10.0.0.0/16 안에서 10.0.1.0/24, 10.0.2.0/24, 10.0.3.0/24... 이런 식으로 subnet을 만들거임.
앞에 3자리는 고정이고 뒤에 8비트만 사용할 수 있음.
-> 2^8 = 256개의 ip를 가질 수 있음.
   * aws에서는 5개 ip가 예약되어 있어서 실제로는 256 - 5 = 251개 사용 가능

-> 만들 수 있는 subnet 개수 : 2^(Subnet Prefix - VPC Prefix)
-> subnet 하나에서 가질 수 있는 ip 개수 : 2^(32bit - Subnet Prefix)

결국은 vpc, subnet, subnet 내부 ip 들은 32비트 안에서 땅따먹기 느낌인듯.
```bash
VPC
10.0.0.0/16
│
├── 10.0.0.0/24
├── 10.0.1.0/24
├── 10.0.2.0/24
├── ...
├── 10.0.254.0/24
└── 10.0.255.0/24
    -> 총 256개     

# /24 subent
# 만들 수 있는 subnet 개수 : 2^(Subnet Prefix - VPC Prefix) = 2^(24-16) = 256개
# subnet 하나에서 가질 수 있는 ip 개수 : 2^(32bit - Subnet Prefix) = 2^(32-24)
10       0        1        0
00001010.00000000.00000001.00000000
 (8비트)   (8비트)   (8비트)   (8비트)
^^^^^^^^.^^^^^^^^.^^^^^^^^.^^^^^^^^
                 | subnet | subnet 안에서 
                 |  개수   | 사용 가능한 ip 개수  
                 |2^8=256 | 2^8=256 -5(aws 예약 ip) = 251  

# /25 subent
# 만들 수 있는 subnet 개수 : 2^(Subnet Prefix - VPC Prefix) = 2^(25-16) = 512개
# subnet 하나에서 가질 수 있는 ip 개수 : 2^(32bit - Subnet Prefix) = 2^(32-25)
10       0        1        0
00001010.00000000.00000001.00000000
 (8비트)   (8비트)   (8비트)   (8비트)
^^^^^^^^.^^^^^^^^.^^^^^^^^.^^^^^^^^
                 | subnet   | subnet 안에서 
                 |  개수     | 사용 가능한 ip 개수  
                 |  2^9=512 | 2^7=128 -5(aws 예약 ip) = 123  
```

| CIDR  |            사용 가능한 IP(대략) | 많이 쓰는 용도                         |
| ----- | -----------------------: | -------------------------------- |
| `/16` |                약 65,536개 | VPC                              |
| `/24` | 256개 (AWS에서는 251개 사용 가능) | Subnet                           |
| `/32` |                       1개 | 특정 IP 하나 지정 (예: 보안 그룹에서 내 PC 허용) |


```bash
CIDR 2진수 공부법!
/16 = VPC에서 자주 사용
/24 = Subnet에서 가장 많이 사용
/25 = /24를 반으로 나눈 것
/26 = /24를 4등분한 것
/27 = /24를 8등분한 것
```
