# IAM 테스트하면서 발생한 문제들

## 1. ec2 내부에서 aws 명령어를 인식 못하는 장애

### 장애 현상
ec2 내부에서 s3로 접근하려고 할 때 aws 명령어 자체를 인식 못하는 상황 발생

### 장애 원인
단순하게 ec2에 aws cli가 설치되어 있지 않은 것.

### 장애 해결
ec2에서 aws cli 설치
ec2 내부에서 aws cli를 계속 써야한다면 user data에 해당 내용을 추가한 런치 템플릿을 새로 만들기
```bash
sudo apt update

sudo apt install awscli -y
```
위 명령어가 안되면 아래 명령어로 설치
```bash
sudo apt install unzip curl -y

curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"

unzip awscliv2.zip

sudo ./aws/install

aws --version
```
